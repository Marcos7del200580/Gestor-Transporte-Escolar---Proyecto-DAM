# Gestor-Transporte-Escolar-Proyecto-DAM 🚍
## Nombre del Proyecto:  (Gestor de Transporte escolar para niños)

## Descripción de nuestro proyecto:

El **Gestor de Transporte Escolar para Niños** es una plataforma digital diseñada para mejorar la seguridad, organización y comunicación en el transporte escolar.

El sistema permitirá a las familias conocer en tiempo real el estado del trayecto de sus hijos, recibir notificaciones relevantes y mantenerse informadas sobre incidencias o cambios en la ruta.

El objetivo principal es aportar **tranquilidad, control y transparencia** en el servicio de transporte escolar, conectando familias, conductores y centros educativos en una misma plataforma.

## Objetivo General:

Desarrolar un aplicación Web que permita:

- Registrar rutas escolares
- Gestionar conductores y vehiculos
- Controlar la subida y bajada de alumnos
- Notificar a los padres en tiempo real
- Mostrar el recorrido del autobus
- Gestionar indidencias
